# Workspace Workforce SDK

Workspace Workforce SDK is een complete projectrepository en documentatieset voor het bouwen, beheren en opschalen van moderne workforce-platformen.

De SDK is bedoeld als centrale bron van waarheid voor een platform waarmee uitzendorganisaties en personeelsintensieve bedrijven hun dagelijkse processen beheren: planning, urenregistratie, werknemers, opdrachtgevers, locaties, documenten, loonstroken, weekstaten, exports, rechten, automatisering en toekomstige AI-ondersteuning.

## Doel

Het doel van deze SDK is om één professioneel fundament te leveren waarmee Workspace Workforce reproduceerbaar gebouwd, uitgebreid en verkocht kan worden.

De SDK ondersteunt:

- Applicatieontwikkeling
- Lovable-builds
- Codex-development
- Supabase-datamodellen
- API-ontwerp
- UI-ontwerp
- Automatiseringen via Make en n8n
- AI-assisted development
- Klantimplementaties
- White-label SaaS-groei

## Hoofdprincipes

- Eerst proces, daarna techniek
- Mobiel en desktop gelijkwaardig
- Rollen en rechten vanaf dag één
- Geen gevoelige documenten publiek beschikbaar
- Database als centrale waarheid
- Auditlogs voor kritieke wijzigingen
- API-first waar integratie nodig is
- Automatisering transparant en herleidbaar
- AI als assistent, niet als oncontroleerbare beslisser
- Modulair uitbreidbaar
- White-label geschikt
- Multi-tenant voorbereid
