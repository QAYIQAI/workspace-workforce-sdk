# n8n Integratie

n8n kan worden gebruikt als self-hosted automation layer.

## Use cases

- Interne workflows
- Periodieke controles
- Documentvalidatie
- API-koppelingen
- Exportverzending
- Klantnotificaties
- Boekhoudintegraties

## Voordelen

- Self-hosted mogelijk
- Meer controle dan SaaS-automation
- Geschikt voor privacygevoelige workflows
- Uitbreidbaar met custom nodes

## Principes

- Credentials veilig beheren
- Workflows versiebeheer geven
- Fouten loggen
- Kritieke acties bevestigen
- Tenantcontext altijd meesturen
