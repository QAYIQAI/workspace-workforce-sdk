# Webhooks

Webhooks verbinden externe systemen met Workspace Workforce.

## Richtlijnen

- Gebruik geheime signature
- Valideer timestamp
- Log payload metadata
- Sla ruwe payload beperkt op
- Voorkom dubbele verwerking
- Retourneer duidelijke statuscodes

## Voorbeelden

- Nieuwe urenregel
- Document ontvangen
- Weekstaat goedgekeurd
- Export klaar
- Klantreactie ontvangen
- Automatisering gestart
