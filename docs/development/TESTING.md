## Acceptatie

Elke release moet minimaal testen:

- Admin kan inloggen.
- Gebruiker zonder recht ziet menu niet.
- Backend blokkeert ongeautoriseerde API.
- Documenten zijn niet publiek toegankelijk.
- Uren kunnen worden ingevoerd en gecontroleerd.
- Weekstaat kan worden gegenereerd.
- Mobiel werkt zonder horizontale overflow.
