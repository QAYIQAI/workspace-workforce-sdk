# Design System

Workspace Workforce gebruikt een zakelijke, rustige en premium SaaS-uitstraling.

## Designprincipes

- Rustig
- Duidelijk
- Professioneel
- Mobielvriendelijk
- Dark mode geschikt
- Compact waar nodig
- Toegankelijk
- Consistent

## Kleurgebruik

Het design moet passen bij een moderne zakelijke applicatie.

Aanbevolen:

- Donkere sidebar
- Lichte contentpanelen
- Subtiele borders
- Duidelijke primaire knop
- Statuskleuren spaarzaam
- Goede contrasten in dark mode

## Typografie

- Heldere sans-serif
- Niet te kleine teksten
- Tabellayout compact maar leesbaar
- Koppen duidelijk hiërarchisch
- Formulierlabels goed zichtbaar

## Spacing

- Geen overdreven hoge rijen
- Compacte tabellen
- Voldoende witruimte in formulieren
- Mobiel minder zijmarge
- Desktop duidelijke contentbreedte

## Dark mode

Dark mode mag geen onleesbare kolommen, badges of tabellen veroorzaken.

Regels:

- Geen zwarte tekst op donkere achtergrond
- Geen felle headerkleur met witte tekst zonder contrasttest
- Borders zichtbaar maar subtiel
- Modals moeten afzonderlijk goed leesbaar zijn
