# UI Guidelines

## Navigatie

Gebruik een duidelijke hoofdstructuur:

- Dashboard
- Planning
- Beheer
- Documenten
- Boekhouding
- Instellingen

## Mobiel

Mobiel gebruikt:

- Drawer menu
- Sticky logout
- Swipebare tabellen
- Compacte filters
- Grote touch targets
- Geen horizontale body-overflow

## Tabellen

Tabellen moeten:

- Compact zijn
- Sorteerbaar waar nodig
- Filterbaar zijn
- Statussen tonen met badges
- Mobiel swipe ondersteunen
- Eerste kolom sticky kunnen maken

## Formulieren

Formulieren moeten:

- Logisch gegroepeerd zijn
- Validatie direct tonen
- Verplichte velden markeren
- Opslaan en annuleren duidelijk tonen
- Geen onnodige velden tonen

## Statussen

Gebruik consistente statussen:

- Open
- Concept
- Ingevoerd
- Gecontroleerd
- Goedgekeurd
- Afgekeurd
- Verzonden
- Geëxporteerd
- Verlopen
