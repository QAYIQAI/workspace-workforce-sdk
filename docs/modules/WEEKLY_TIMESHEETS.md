# Weekstaten

Weekstaten bundelen uren per week.

## Functies

- Weekstaat genereren
- Totalen berekenen
- PDF maken
- Klantgoedkeuring voorbereiden
- Verzenden
- Exporteren
- Heropenen

## Groepering

Weekstaten kunnen worden gegroepeerd per:

- Opdrachtgever
- Locatie
- Werknemer
- Weeknummer
- Jaar

## Statussen

- open
- compleet
- gecontroleerd
- goedgekeurd
- verzonden
- geëxporteerd
- heropend
