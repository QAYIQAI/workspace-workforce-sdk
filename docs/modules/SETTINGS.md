# Instellingen

De instellingenmodule beheert configuratie.

## Onderdelen

- Organisatiegegevens
- Gebruikers
- Rollen
- Menu-rechten
- Documenttypes
- Exportinstellingen
- E-mailtemplates
- Thema
- Taal
- Automatiseringen

## Principes

- Alleen beheerders mogen kritieke instellingen aanpassen.
- Wijzigingen worden gelogd.
- Instellingen zijn tenant-aware.
