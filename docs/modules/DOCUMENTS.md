# Documenten

De documentenmodule beheert personeelsdocumenten.

## Functies

- Document uploaden
- Document vervangen
- Document downloaden
- Geldigheid vastleggen
- Verlopen documenten signaleren
- Documenttype beheren
- Status controleren
- Auditlog bij download en wijziging

## Toegestane bestandstypen

- PDF
- PNG
- JPG
- JPEG

## Beveiliging

- Geen publieke storage
- Download via beveiligde route
- Rechtencontrole
- Bestandstypevalidatie
- Bestandsgroottebeperking
