# Locaties

De locatiemodule beheert werkplekken van opdrachtgevers.

## Functies

- Locatie aanmaken
- Adres vastleggen
- Contactpersoon vastleggen
- Actieve status beheren
- Werknemers koppelen
- Planning per locatie bekijken

## Relaties

- Locatie hoort bij opdrachtgever
- Locatie heeft planning
- Locatie heeft uren
- Locatie kan contractafspraken hebben
