# Loonstroken

De loonstrokenmodule beheert loonstrookbestanden per werknemer.

## Functies

- Loonstrook uploaden
- Loonstrook koppelen aan periode
- Downloaden door bevoegde gebruiker
- Verzending loggen
- Historie bewaren

## Velden

- employee_id
- period
- year
- file_id
- uploaded_by
- uploaded_at
- status
