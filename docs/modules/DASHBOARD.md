# Dashboard

Het dashboard toont operationele kerninformatie.

## KPI’s

- Actieve werknemers
- Actieve opdrachtgevers
- Planning vandaag
- Ontbrekende uren
- Open weekstaten
- Verlopen documenten
- Exports klaar
- Laatste acties

## Doel

Het dashboard moet managers en administratie direct inzicht geven in wat aandacht nodig heeft.
