# Boekhoudingsexport

De boekhoudingsexportmodule bereidt gegevens voor externe administratie voor.

## Exporttypes

- Weekuren
- Maanduren
- Facturatievoorbereiding
- Loonvoorbereiding
- Documentcontrole
- Correctieoverzicht

## Formaten

- CSV
- XLSX
- PDF
- JSON via API

## Principes

- Export als batch
- Exportstatus bewaren
- Herexport mogelijk
- Verzending loggen
- Geëxporteerde regels vergrendelen waar nodig
