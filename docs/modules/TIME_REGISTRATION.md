# Urenregistratie

De urenregistratiemodule registreert gewerkte uren.

## Functies

- Uren per dag invoeren
- Start- en eindtijd registreren
- Pauze registreren
- Opmerkingen toevoegen
- Afwijkingen vastleggen
- Uren controleren
- Uren goedkeuren
- Uren heropenen
- Uren exporteren

## Berekening

total_hours = eindtijd - starttijd - pauze

## Statussen

- concept
- ingevoerd
- gecontroleerd
- goedgekeurd
- afgekeurd
- geëxporteerd
- heropend
