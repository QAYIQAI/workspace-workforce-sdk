# Backup en Monitoring

## Back-ups

Back-up van:

- Database
- Documentstorage
- Exports
- Configuratie
- Belangrijke logs

## Frequentie

- Dagelijkse databaseback-up
- Dagelijkse storageback-up

## Monitoring

Monitor:

- Uptime

- Database performance
- Storagegebruik
- Login failures
- Export failures
- Automatiseringsruns
- AI-verbruik
- E-mailverzending

## Incidenten

Elke storing krijgt:

- Tijdstip
- Impact
- Oorzaak
- Oplossing
- Preventieve actie
